Cerita Misi

Taklimat Agent

Selamat datang, Agent. Sebuah organisasi telah menerima komputer server baharu yang akan digunakan sebagai pusat perkhidmatan rangkaian. Sebelum Active Directory, DNS dan DHCP boleh dikonfigurasikan, sistem operasi Windows Server 2019 mesti dipasang dengan betul.

Tugas anda ialah menyediakan media pemasangan, memulakan proses boot, memilih edisi yang sesuai, memasang sistem operasi dan memastikan server berjaya memasuki Server Manager tanpa sebarang ralat.

Nota Ringkas

Kenali Windows Server 2019

Sistem Operasi Server

Direka untuk mengurus pengguna, komputer, fail, aplikasi dan perkhidmatan rangkaian secara berpusat.

Keselamatan

Menyediakan kawalan akses, polisi keselamatan dan pengurusan identiti organisasi.

Peranan Server

Menyokong Active Directory, DNS, DHCP, File Server dan Hyper-V.

Prosedur

Langkah Pemasangan

1

Sediakan fail ISO

Gunakan fail ISO Windows Server 2019 yang sah dan pastikan fail tidak rosak.

2

Cipta USB bootable

Gunakan Rufus atau Ventoy untuk menyalin fail ISO ke dalam USB.

3

Tetapkan Boot Priority

Masuk ke BIOS atau UEFI dan pilih USB sebagai peranti boot pertama.

4

Pilih bahasa dan papan kekunci

Tetapkan bahasa, format masa dan jenis papan kekunci sebelum menekan Install now.

5

Pilih edisi Desktop Experience

Pilih Windows Server 2019 Standard atau Datacenter dengan pilihan Desktop Experience.

6

Pilih Custom Installation

Pilih pemasangan baharu dan tentukan partition storan yang akan digunakan.

7

Tunggu pemasangan selesai

Sistem akan menyalin fail, memasang ciri dan memulakan semula komputer beberapa kali.

8

Tetapkan kata laluan Administrator

Gunakan kata laluan yang mengandungi huruf besar, huruf kecil, nombor dan simbol.

9

Log masuk dan buka Server Manager

Pastikan desktop berjaya dipaparkan dan Server Manager dibuka tanpa ralat.

💡

Tip Penting

Pilih edisi yang mempunyai label Desktop Experience jika anda mahu menggunakan paparan grafik. Pilihan tanpa label tersebut akan memasang Server Core yang menggunakan command line.
