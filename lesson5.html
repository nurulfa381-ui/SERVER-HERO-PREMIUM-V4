Taklimat Agent

Domain serverhero.local telah berfungsi. Organisasi kini mahu akaun pekerja disusun mengikut jabatan supaya pengurusan pengguna dan polisi keselamatan menjadi lebih kemas.

Tugas anda ialah mencipta OU bernama IT Department, mencipta pengguna bernama Ali dengan username ali, menetapkan kata laluan P@ssw0rd123 dan mengaktifkan pilihan User must change password at next logon.

Nota Ringkas

Objek Penting dalam Active Directory

Organizational Unit

Bekas logik untuk menyusun pengguna, komputer dan kumpulan mengikut jabatan.

User Object

Akaun yang membolehkan seseorang log masuk ke domain.

Group Object

Digunakan untuk mengurus kebenaran beberapa pengguna secara serentak.

Computer Object

Mewakili komputer yang menjadi ahli domain.

Password Policy

Menentukan panjang, kerumitan dan tempoh kata laluan.

Account Status

Akaun boleh diaktifkan, dinyahaktifkan, dikunci atau dipadam.

Prosedur

Langkah Mencipta OU dan Pengguna

1

Buka Active Directory Users and Computers

Dalam Server Manager, pilih Tools dan buka konsol ADUC.

2

Cipta Organizational Unit

Klik kanan domain, pilih New, Organizational Unit dan masukkan IT Department.

3

Cipta Pengguna Baharu

Klik kanan OU, pilih New dan kemudian User.

4

Masukkan Maklumat Pengguna

Masukkan nama Ali dan logon name ali.

5

Tetapkan Kata Laluan

Gunakan kata laluan kukuh seperti P@ssw0rd123.

6

Paksa Tukar Kata Laluan

Tandakan User must change password at next logon.

7

Sahkan Akaun

Pastikan pengguna Ali berada dalam OU IT Department dan akaun berstatus Enabled.

💡

Tip Penting

Gunakan OU untuk memudahkan pelaksanaan Group Policy, delegasi tugas pentadbiran dan pengurusan objek domain.
