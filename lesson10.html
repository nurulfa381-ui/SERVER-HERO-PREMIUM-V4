Cerita Misi

Taklimat Agent

Organisasi mahu semua pengguna domain mencetak melalui satu pencetak rangkaian yang diurus secara berpusat.

Cipta Office Printer menggunakan IP 192.168.10.50, kongsi sebagai OFFICE-PRINTER dan uji melalui \\SERVER\OFFICE-PRINTER.

Nota Ringkas

Konsep Penting Print Server

Print Server

Mengurus pencetak, driver, queue dan permission.

TCP/IP Port

Menghubungkan server kepada printer melalui IP.

Print Queue

Senarai tugasan cetakan yang menunggu.

Printer Driver

Perisian komunikasi antara Windows dan printer.

Printer Sharing

Membolehkan client menggunakan printer rangkaian.

Permission

Mengawal Print, Manage Documents dan Manage Printer.
