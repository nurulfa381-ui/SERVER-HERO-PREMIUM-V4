Windows Server Backup dan Restore

Pelajari cara memasang Windows Server Backup, mencipta backup schedule, memilih lokasi backup dan memulihkan data yang hilang.

Cerita Misi

Taklimat Agent

Server organisasi menyimpan konfigurasi domain dan fail penting. Risiko kegagalan cakera, serangan malware atau kesilapan pengguna boleh menyebabkan kehilangan data.

Tugas anda ialah menyediakan backup harian pada jam 10:00 malam, memilih Full Server Backup, menyimpan backup pada pemacu E: dan menguji pemulihan folder D:\IT-Share.

Nota Ringkas

Konsep Penting Backup

Full Backup

Menyalin keseluruhan data dan konfigurasi server.

Backup Schedule

Menjalankan backup secara automatik mengikut masa ditetapkan.

Backup Destination

Lokasi fizikal atau rangkaian untuk menyimpan backup.

Restore

Memulihkan fail, folder atau sistem daripada backup.

System State

Menyimpan komponen kritikal seperti registry dan Active Directory.

Verification

Mengesahkan backup boleh digunakan ketika pemulihan.

Prosedur

Langkah Backup dan Restore

1

Pasang Windows Server Backup

Gunakan Add Roles and Features dan pilih Windows Server Backup.

2

Buka Windows Server Backup

Pilih Local Backup pada panel kiri.

3

Pilih Backup Schedule

Tetapkan backup dijalankan setiap hari.

4

Pilih Full Server

Pastikan semua volume dan komponen kritikal disertakan.

5

Tetapkan Masa

Gunakan masa 10:00 malam.

6

Pilih Lokasi Backup

Simpan backup pada pemacu E:.

7

Jalankan Backup

Pastikan status menunjukkan Completed Successfully.

8

Uji Restore

Pulihkan folder D:\IT-Share dan semak kandungannya.

⚠️

Tip Penting

Backup mesti disimpan pada pemacu atau lokasi berasingan daripada data asal. Backup pada cakera yang sama tidak melindungi daripada kegagalan cakera.
