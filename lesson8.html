Cerita Misi

Taklimat Agent

Komputer client kini telah menyertai domain, tetapi alamat IP masih perlu ditetapkan secara manual. Organisasi mahu server memberikan alamat IP secara automatik kepada semua client.

Tugas anda ialah memasang DHCP Server, mencipta Scope SERVERHERO-LAN, menetapkan julat IP, Gateway, DNS dan mengesahkan client menerima alamat IP melalui arahan ipconfig /renew.

Nota Ringkas

Konsep Penting DHCP

DHCP Server

Memberikan alamat IP dan tetapan rangkaian secara automatik.

Scope

Julat alamat IP yang boleh diberikan kepada client.

Exclusion Range

Alamat IP yang tidak boleh diberikan kepada client.

Lease Duration

Tempoh client dibenarkan menggunakan alamat IP.

Option 003

Menetapkan Default Gateway untuk client.

Option 006

Menetapkan DNS Server untuk client.

Prosedur

Langkah Konfigurasi DHCP

1

Pasang DHCP Server Role

Gunakan Add Roles and Features dalam Server Manager.

2

Complete DHCP Configuration

Authorize server menggunakan akaun domain.

3

Buka DHCP Manager

Pilih IPv4 dan cipta New Scope.

4

Masukkan Scope Name

Gunakan nama SERVERHERO-LAN.

5

Tetapkan IP Range

Gunakan 192.168.10.100 hingga 192.168.10.200.

6

Tetapkan Exclusion Range

Kecualikan 192.168.10.100 hingga 192.168.10.110.

7

Konfigurasi Router dan DNS

Gateway 192.168.10.1 dan DNS 192.168.10.10.

8

Aktifkan Scope

Pastikan Scope berstatus Active.

9

Uji Client

Jalankan ipconfig /release dan ipconfig /renew.

💡

Tip Penting

Jangan masukkan alamat IP server, router atau printer dalam julat yang boleh diberikan kepada client. Gunakan Exclusion Range untuk melindungi alamat penting.
