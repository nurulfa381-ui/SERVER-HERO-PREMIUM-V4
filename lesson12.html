Cerita Misi

Taklimat Agent

Server organisasi kini beroperasi sepenuhnya. Namun, serangan malware, akses tanpa kebenaran dan kegagalan prestasi boleh mengganggu operasi.

Tugas anda ialah mengaktifkan Windows Defender Firewall, menjalankan antivirus scan, memasang kemas kini keselamatan, menyemak Event Viewer dan memantau CPU menggunakan Performance Monitor.

Nota Ringkas

Komponen Keselamatan dan Monitoring

Windows Firewall

Mengawal trafik masuk dan keluar berdasarkan peraturan keselamatan.

Defender Antivirus

Mengesan dan menghapuskan malware.

Windows Update

Memasang patch keselamatan dan pembaikan sistem.

Event Viewer

Memaparkan log sistem, keselamatan dan aplikasi.

Performance Monitor

Memantau CPU, memori, cakera dan rangkaian.

UAC

Mengurangkan perubahan sistem tanpa kebenaran.

Prosedur

Langkah Keselamatan dan Monitoring

1

Aktifkan Firewall

Pastikan Domain Profile berstatus On.

2

Jalankan Full Scan

Gunakan Windows Defender Antivirus.

3

Pasang Windows Update

Pastikan semua kemas kini kritikal berjaya dipasang.

4

Semak Event Viewer

Periksa Windows Logs, System dan Security.

5

Analisis Event ID

Kenal pasti ralat kritikal dan amaran keselamatan.

6

Buka Performance Monitor

Tambah counter Processor, Memory dan Disk.

7

Semak CPU Usage

Pastikan penggunaan CPU berada pada tahap normal.

8

Dokumentasikan Hasil

Catat status keselamatan dan tindakan pembetulan.

⚠️

Tip Penting

Jangan tutup firewall untuk menyelesaikan masalah rangkaian. Cipta inbound rule yang betul dan semak port yang diperlukan.
