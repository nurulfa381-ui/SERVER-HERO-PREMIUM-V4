Cerita Misi

Taklimat Agent

Organisasi memerlukan satu lokasi pusat untuk menyimpan dokumen jabatan. Pengguna dalam OU IT Department perlu boleh membaca dan mengubah fail, tetapi pengguna lain hanya dibenarkan membaca.

Tugas anda ialah menyediakan folder IT-Share, mengaktifkan perkongsian rangkaian, menetapkan permission yang betul dan menguji akses menggunakan laluan \\SERVER\IT-Share.

Nota Ringkas

Konsep Penting File Sharing

File Server

Server yang menyimpan dan berkongsi fail kepada pengguna rangkaian.

Shared Folder

Folder yang boleh diakses melalui rangkaian.

NTFS Permission

Mengawal akses fail dan folder pada sistem fail NTFS.

Share Permission

Mengawal akses apabila folder dibuka melalui rangkaian.

Read

Membenarkan pengguna melihat dan membuka fail.

Modify

Membenarkan pengguna membaca, menambah, mengubah dan memadam fail.

Prosedur

Langkah Konfigurasi File Server

1

Pasang File Server Role

Gunakan Add Roles and Features dalam Server Manager.

2

Cipta Folder

Cipta folder D:\IT-Share.

3

Aktifkan Sharing

Buka Properties, Sharing dan Advanced Sharing.

4

Tetapkan Share Name

Gunakan nama IT-Share.

5

Tetapkan Share Permission

Berikan Change dan Read kepada IT Users.

6

Tetapkan NTFS Permission

Berikan Modify kepada kumpulan IT Users.

7

Uji Laluan UNC

Akses menggunakan \\SERVER\IT-Share.

8

Uji Cipta Fail

Pastikan pengguna yang dibenarkan boleh mencipta dan mengubah fail.

⚠️

Tip Penting

Akses sebenar ditentukan oleh gabungan Share Permission dan NTFS Permission. Permission yang paling ketat akan digunakan.
